# MLMUPC Administrative Codes

This repository contains administrative codes data for Cambodia's administrative divisions.

## Files

- `MLMUPC.csv` - Administrative codes data in CSV format
- `MLMUPC.txt` - Administrative codes data in text format

## Data Structure

The data contains hierarchical administrative codes for:
- Provinces
- Districts
- Communes
- Villages

Each entry includes:
- Administrative codes
- Khmer names (name_kh)
- English names (name_en)
- Hierarchical relationships

## Usage

This data can be used for:
- Administrative code lookups
- Geographic data processing
- Database imports
- Administrative boundary mapping

## Source

Data extracted from official Cambodian administrative sources.
